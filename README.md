# Calculadora React Native

Aplicativo de calculadora mobile desenvolvido em **React Native**

## Requisitos e Funcionalidades Implementadas

* **Operações Básicas:** Suporte a Adição, Subtração, Multiplicação e Divisão.
* **Raiz Quadrada:** Função de cálculo da raiz quadrada.
* **Botão de Limpeza:** Botão dedicado para zerar o visor e limpar o armazenamento da calculadora.

## Tecnologias Utilizadas

* React Native
* Expo
* JavaScript
